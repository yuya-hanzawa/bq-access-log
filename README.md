## Usage
### 1. Clone Codes
```
$ git clone https://github.com/zawa1120/bq-access-log.git
```

### 2. install Google Cloud SDK
```
$ brew install google-cloud-sdk --cask
```

### 3. 
```
$ gcloud auth login
```

## Requirements
- macOS Big Sur 11.4 Apple M1
  - Homebrew 3.3.6

- Google Cloud SDK 365.0.1
  - bq 2.0.71
  - core 2021.11.19
  - gsutil 5.5

- Tools
  - Terraform v1.0.11
